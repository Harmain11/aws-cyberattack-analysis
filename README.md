# AWS Cyberattack Analysis

## Overview
This notebook analyzes a large dataset of cyberattack logs collected from Amazon Web Services (AWS) honeypots. It performs data cleaning, exploratory data analysis, and visualizations on attack patterns, including geolocation mapping of attack origins, protocol distributions, and attack frequency by host and time.

## Features
- Data cleaning and handling of missing values
- Extraction of datetime features for temporal analysis
- Visualization of protocol distributions using bar and pie charts
- Analysis of attacks by month, hour, host, and country
- Geographical visualization of cyberattacks using heat maps
- Identification of top attack sources
- Basic data preprocessing techniques for clustering and machine learning

## Tech Stack
- Python
- Jupyter Notebook / Google Colab
- Libraries: pandas, numpy, matplotlib, seaborn, folium, scikit-learn

## How to Use
1. Clone or download the repository.
2. Open the notebook `aws-cyberattack-analysis.ipynb` in Jupyter Notebook or Google Colab.
3. Upload the dataset `AWS_Honeypot.csv` to the working directory or Colab environment.
4. Run the notebook cells sequentially to perform analysis and generate visualizations.
